# 🎓 School Management System (C++ + MySQL)

A simple console-based system developed in C++ using OOP and MySQL database.

## 🚀 Features

🔐 Login / Signup
➕ Add Student
🔍 Search Student
❌ Delete Record
✏️ Update Record
📋 Show All Data
🔢 Count Students

## 🛠️ Tech Used

C++ | OOP | MySQL

## ⚙️ Setup

1. Import database (school.sql)
2. Update MySQL username & password in code
3. Compile & Run
